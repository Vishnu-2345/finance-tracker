# Finance Tracker

A Full Stack Finance Tracker application built using React, FastAPI, and MySQL.

## Features

* User Registration & Login
* JWT Authentication
* Add Transactions
* Edit Transactions
* Delete Transactions
* Dynamic Balance Calculation
* Income & Expense Tracking
* Real-Time Pie Charts
* Expense Analytics Dashboard
* Responsive UI

## Tech Stack

### Frontend

* React.js
* Tailwind CSS
* Recharts

### Backend

* FastAPI
* SQLAlchemy
* Passlib
* JWT Authentication

### Database

* MySQL

## How To Run

### Backend

```bash
cd backend
uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm run dev
```
